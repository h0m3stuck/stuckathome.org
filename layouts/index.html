Portfolio

Akira

A privacy-friendly personal assistant, written in Python 3.

Features

  • Wakeword detection using an open-source machine-learning algorithm
  • Speech recognition using Google Speech APIs
  • Simple, modular API for adding new keywords and skills

Litcoin

An incentivized peer review system using an Ethereum token.

Features

  • Uses ERC20 token LIT as payment
  • Off-chain storage through IPFS
  • Deployed on Rinkeby testnet for public usage

This project won 3rd place at HackNEHS.

This project is continued at Scintillating.

Duck Feed

A cross-platform RSS feed discoverer and reader, using DuckDuckGo to provide search results.

Features

  • Scrapes DuckDuckGo searches in the background for privacy-friendly feed discovery
  • Uses parallelism to find results faster
  • Has a built-in WebKit engine to show RSS entries as full HTML and play back video feeds

Quecoin

A distributed, incentivized Q&A system running on the Ethereum blockchain and using an ERC20 token.

Features

  • Uncensorable, community-curated question asking and answering through an objective voting system
  • Uses QUE, an ERC20 token, as staking to incentivize good behavior
  • Total decentralization via Solidity smart contract logic and (planned) off-chain storage through IPFS

This project won 3rd place in the experienced divison of MIT Blueprint 2018.

This project is continued at Scintillating.

About Us

We are a team of seven high school sophomores and juniors who have gone to various hackathons, including MetroHacks, MIT Blueprint 2017, HackNEHS, LexHack, and MIT Blueprint 2018.

We won second place for Akira at LexHack, third place at HackNEHS for Litcoin, and third place at MIT Blueprint 2018 for Quecoin. We are continuing our crypto projects under the Scintillating company.
